이게 뭐에요?
==========

트위터에선 자신이 관심있어하는 주제에 대한 트윗을 모아 보여주는 토픽기능을 제공합니다. 하지만 트위터의 토픽목록에선 자신이 사용하는 언어의 토픽만 보여주며 다른 언어로 된 토픽을 팔로우하는 기능이 없습니다. 그래서 다른 언어의 토픽을 찾고 팔로우할 수 있도록 토픽을 모아봤습니다.
